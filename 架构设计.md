## 架构目标

本裸机系统是基于 system_tick + protothreads + 状态机 + 前后台 +clib 的 IOT 系统框架:

- 支持 tickless 操作，所以 MCU 最好有 LPTimer 支持
- 支持 broker event



## 框架层

- APP
- components
- device
- driver
- hal
- sdk hal



## APP 实现



```c
/**main.c */

void main(void)
{
    param_init();
    bsp_init();
    while(1)
    {
        try_broker_event(context)
        if(emergency_event) {
            emergency_task(context);
        }
        bare_task_a(context);
        bare_task_b(contex);
        
        if(all task idle / state machine off)
        {
            mcu_deep_sleep
        } 
        
        if(tickless) {
            mcu_stop
        }
    }
    
}

```
